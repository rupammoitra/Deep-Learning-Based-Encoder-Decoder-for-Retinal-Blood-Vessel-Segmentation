Deep Learning-Based Encoder–Decoder for Retinal Blood Vessel Segmentation
Project Overview
This project focuses on retinal blood vessel segmentation using a deep learning-based encoder–decoder approach. The objective is to accurately segment retinal blood vessels from fundus images for retinal image analysis and diabetic retinopathy screening.

Features
Retinal blood vessel segmentation
Image preprocessing
Blood vessel extraction
Performance evaluation
Visualization of segmentation results
